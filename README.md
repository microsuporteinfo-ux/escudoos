# EscudoOS - Sistema de Gestão para Segurança Eletrônica

## Descrição
Sistema completo de gerenciamento de ordens de serviço para empresas de segurança eletrônica.

## Funcionalidades
- ✅ Autenticação multi-perfil (Admin, Técnico, Cliente)
- ✅ Gestão completa de clientes
- ✅ Cadastro de produtos e serviços
- ✅ Controle de estoque com baixa automática
- ✅ Ordens de serviço completas com laudos técnicos
- ✅ Agendamentos e calendário
- ✅ Relatórios avançados (PDF, Excel, Google Sheets)
- ✅ Integração com WhatsApp (Twilio)
- ✅ Integração com Google Drive
- ✅ Sistema de assinaturas (Stripe)
- ✅ Geração de PDF com assinatura digital
- ✅ Upload de fotos e documentos
- ✅ Dashboard com métricas
- ✅ Responsivo e moderno

## Requisitos
- PHP 8.0 ou superior
- Composer 2.0 ou superior
- Node.js 14.0 ou superior
- MySQL 5.7 ou superior
- Servidor web (Apache/Nginx)
